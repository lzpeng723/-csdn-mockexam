# 考试说明

- C1 见习工程师能力认证，需通过C1认证才能参加C4或者C5的认证考试
- C4 专项工程师能力认证，分为三个方向各有两个级别，需按方向逐级参加认证考试
- C5 全栈工程师认证，分为三个级别，需逐级参加认证考试
